# ☕ estudos-java

Este repositório armazena todos os exercícios práticos e códigos desenvolvidos durante as minhas aulas de Java.

## 📚 Conteúdo Estudado

* **Fundamentos do Java**: Sintaxe básica, variáveis e operadores.
* **Lógica de Programação**: Estruturas condicionais e laços de repetição.
* **Estruturas de Dados**: Manipulação de arrays, vetores e lógica de inversão (como a sequência invertida).

## 🛠️ Tecnologias Utilizadas

* **Java**
* **Git e GitHub**

## 🚀 Como Executar os Projetos

1. **Clone o repositório:**
   ```bash
   git clone https://github.com
   ```
2. **Compile o arquivo desejado:**
   ```bash
   javac NomeDoArquivo.java
   ```
3. **Execute o programa:**
   ```bash
   java NomeDoArquivo
   ```
